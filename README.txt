Pour lancer mon application, il faut installer toutes les tables situées à l'adresse suivante :
https://github.com/pawlodkowski/northwind_data_clean/tree/master.

Pour continuer, il suffit simplement de lancer la commande :
node .\server.js
puis d'ouvrir un navigateur à l'adresse suivante :
http://localhost:3000,
ou d'accéder directement au lien indiqué dans le message affiché dans la console.

Mon application permet de récupérer des données intéressantes sans entrer dans trop de détails.
L'objectif est de suivre l'évolution des données et d'en faire des statistiques.
